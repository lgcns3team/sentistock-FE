# 🎨 프로젝트 개요
<div align=center>
<img width="1000" alt="프로젝트소개" src="https://github.com/user-attachments/assets/cd8b89cf-d90b-40a6-bfd6-c2cadda8ad93" />
<br></br>	
</div>

주식 시장은 정치·경제 이벤트, 사회 이슈, 투자 심리 등 다양한 요인이 동시에 작용해 변동성이 크고 예측이 어렵습니다.

본 프로젝트는 이러한 한계를 전제로, **뉴스에 담긴 시장의 감정(Sentiment)** 이 주가 변동의 사전 신호로 활용될 수 있다는 문제의식에서 출발했습니다.

센티스톡(SentiStock)은 금융 뉴스 데이터를 기반으로 **섹터 및 종목별 시장 감정을 수치화**하고, 이를 통해 투자자가 시장 분위기와 심리 변화를 **직관적으로 인식**할 수 있도록 돕는 투자 판단 보조 서비스입니다.

주가 예측이나 매매 신호를 직접 제공하기보다, **시장 변화의 사전 흐름을 파악할 수 있는 감정 지표**를 제공하여 감정적 매매를 완화하고 합리적인 의사결정을 지원하는 것을 목표로 합니다.

# 🌟 기대효과

- **뉴스 기반 감정 지표의 실효성 검증**
    
    백테스트 결과, 뉴스 감정 점수는 시장 참여자의 **단기 심리 변화를 반영하는 유의미한 신호**로 활용 가능함을 확인했습니다.
    
- **직관적인 투자 판단 보조**
    
    감정 데이터는 복잡한 재무 지표를 대체하기보다, 투자자가 **시장 분위기를 직관적으로 인식**할 수 있도록 돕는 보조 정보로 작동합니다.
    
- **감정적 매매 완화 및 리스크 회피 지원**
    
    감정 점수 변화와 수익 구간 도달 시점을 **의사결정 트리거**로 제공하여,
    
    투자 판단을 자동화하지 않으면서도 감정적 매매를 줄이고 **손실 가능성이 높은 구간 회피**를 지원합니다.
    
- **확장 가능한 투자 보조 서비스**
    
    SentiStock은 감정 데이터를 기반으로 한 **투자 판단 보조 서비스로의 확장 가능성**을 제시합니다.

# 👥 Convention

### Branch Naming Convention 🪵

| 머릿말  | 설명                               |
| ------- | ---------------------------------- |
| main    | 서비스 브랜치                      |
| develop | 배포 전 작업 기준                  |
| feat    | 기능 단위 구현                     |
| hotfix  | 서비스 중 긴급 수정 건에 대한 처리 |

### Commit Convention 🚥
| 머릿말   | 설명                                                 |
| -------- | ------------------------------------------------------|
| feat | 새로운 기능에 대한 커밋 |
| fix | 버그 수정에 대한 커밋 |
| build | 빌드 관련 파일 수정 / 모듈 설치 또는 삭제에 대한 커밋 |
| chore | 그 외 자잘한 수정에 대한 커밋 |
| ci | ci 관련 설정 수정에 대한 커밋 |
| docs | 문서 수정에 대한 커밋 |
| style | 코드 스타일 혹은 포맷 등에 관한 커밋 |
| refactor | 코드 리팩토링에 대한 커밋 |
| test | 테스트 코드 수정에 대한 커밋 |
| perf | 성능 개선에 대한 커밋 |
| save | 본인 저장용 커밋(리뷰 필요 x) |
| delete | 삭제 내용 |
